# Spectrum Example

![Picture](../../assets/example_spectrum.png)

An example showcasing an audio-spectrum analyzer trying to mimic the style of a
modern VST-plugin.

## Files

| Path       | Description                           |
| ---------- | ------------------------------------- |
| `main.rs`  | Application logic and curve rendering |
| `math.rs`  | Math helpers and utilities            |
| `audio.rs` | Device fetching, sampling, etc.       |
