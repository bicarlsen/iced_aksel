# Candlestick example

![Picture](../../assets/example_candlestick.png)

An example trying to mimic the style of the charts within TradingView.

## Files

| Path            | Description                    |
| --------------- | ------------------------------ |
| `main.rs`       | Application logic              |
| `chart.rs`      | Chart logic                    |
| `items.rs`      | Item definitions and rendering |
| `indicators.rs` | Math for SMA and Boiler Bands  |
